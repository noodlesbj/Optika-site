# Optics Educational Platform

Образовательная платформа по физике (раздел Оптика) на базе Django. Проект поддерживает два языка: русский и казахский.

## Инструкция по запуску

Выполните следующие шаги для запуска проекта на вашем локальном компьютере:

### 1. Автоматическая настройка

В проекте есть скрипт автоматической установки, который установит Django, создаст базу данных и загрузит все необходимые данные:

```bash
python setup.py
```

### 2. Ручная настройка (если скрипт не сработал)

Если автоматический скрипт завершился с ошибкой, выполните следующие команды по очереди:

```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Создание миграций для базы данных
python manage.py makemigrations core
python manage.py migrate

# 3. Загрузка данных (10 тем и вопросы для тестов)
python manage.py loaddata core/fixtures/initial_data.json
```

### 3. Запуск сервера

После успешной настройки запустите сервер разработчика:

```bash
python manage.py runserver
```

Откройте в браузере: `http://127.0.0.1:8000/`

## Функционал

- Двуязычный интерфейс (RU / ҚАЗ)
- 10 тем по оптике с лекционным материалом
- Тесты по каждой теме (5 вопросов)
- Регистрация и авторизация пользователей
- Профиль с историей прохождения тестов
- Современный и адаптивный дизайн
